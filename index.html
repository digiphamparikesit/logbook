Pengaturan Sinkronisasi Google Sheets (Database Utama)

Aplikasi ini membaca dan menulis langsung ke Google Sheets. URL Web App sudah ditanam sebagai bawaan (default) di dalam kode, jadi aplikasi ini otomatis terhubung di perangkat mana pun tanpa perlu tempel URL lagi. Ubah kolom di bawah hanya jika Anda ingin memakai spreadsheet lain.

  1. Buka Google Sheets baru (boleh kosong).
  2. Menu Extensions → Apps Script.
  3. Hapus kode default, tempel kode di bawah ini.
  4. Klik Deploy → New deployment → Web app. Pilih "Execute as: Me" dan "Who has access: Anyone". (Kalau sudah pernah deploy sebelumnya, pilih "Manage deployments" → edit → naikkan versi baru.)
  5. Salin URL Web App, tempel ke kolom di bawah, klik Simpan (opsional — hanya jika ganti spreadsheet).
